# Superstore Sales Dashboard - Task 1

## 📊 Project Overview
An interactive Excel Sales Dashboard built using the Superstore Sales Dataset as part of Data Analyst Internship at SkillCraft Technology.

## 🛠️ Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Graphs
- Conditional Formatting

## 📁 Dataset
- Source: Superstore Sales Dataset (train.csv)
- Records: 9800+ rows of sales data
- Fields: Order ID, Category, Region, Segment, Sales, etc.

## 📈 Dashboard Features
- Sales by Category (Column Chart)
- Sales by Region (Column Chart)
- Sales Trend by Year (Line Chart)
- Sales by Segment (Pie Chart)
- Conditional Formatting to highlight high/low sales

## 🔍 Key Insights
- Technology is the highest selling category
- Sales show consistent growth from 2015 to 2018
- Consumer segment contributes the most to total sales
